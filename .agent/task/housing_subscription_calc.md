# 주택 청약 가점 계산기 개발 가이드

## 1. 개요
주택 청약 시 당첨자 선정을 위한 가점제 점수를 계산하는 시뮬레이터입니다.
무주택 기간, 부양가족 수, 청약통장 가입 기간을 기반으로 총 84점 만점 중 사용자의 점수를 산출합니다.

## 2. 입력 항목 (Input)

| 항목 | 입력 타입 | 단위 | 비고 |
|---|---|---|---|
| **무주택 기간** | Select Box 또는 Number | 년 (기간) | 만 30세 이상 또는 혼인신고일 기준 |
| **부양가족 수** | Number | 명 | 본인 제외 |
| **청약통장 가입 기간** | Select Box 또는 Number | 년 (기간) | 가입일 기준 |

## 3. 계산 로직 (Logic)

**총점 = ① 무주택 기간 점수 + ② 부양가족 점수 + ③ 입주자 저축 가입 기간 점수**

### ① 무주택 기간 (32점 만점)
*   **기본 로직:** 1년 미만 2점부터 시작하여 1년마다 2점씩 증가
*   30세 미만 미혼자: 0점
*   15년 이상: 32점 (상한)

| 기간 | 점수 |
|---|---|
| 1년 미만 | 2점 |
| 1년 이상 ~ 2년 미만 | 4점 |
| ... (매년 +2점) | ... |
| 15년 이상 | 32점 |

### ② 부양가족 수 (35점 만점)
*   **기본 로직:** 0명 5점부터 시작하여 1명당 5점씩 증가
*   6명 이상: 35점 (상한)

| 가족 수 | 점수 |
|---|---|
| 0명 | 5점 |
| 1명 | 10점 |
| ... (1명당 +5점) | ... |
| 6명 이상 | 35점 |

### ③ 청약통장 가입 기간 (17점 만점)
*   **기본 로직:** 6개월 미만 1점, 이후 1년마다 1점씩 증가 (단, 6개월~1년 구간 별도)
*   15년 이상: 17점 (상한)

| 기간 | 점수 |
|---|---|
| 6개월 미만 | 1점 |
| 6개월 이상 ~ 1년 미만 | 2점 |
| 1년 이상 ~ 2년 미만 | 3점 |
| ... (매년 +1점) | ... |
| 15년 이상 | 17점 |

## 4. 구현 시 고려사항
*   **무주택 기간 산정의 복잡성:** 단순 년수 입력이 아니라, 사용자가 '생년월일', '혼인여부', '무주택 시작일' 등을 입력하면 자동으로 기간을 계산해주는 '상세 모드'를 추가하면 사용자 편의성이 높아집니다.
*   **검증:** 각 항목의 합계가 84점을 초과하지 않도록 로직을 점검해야 합니다.

## 5. 결과 출력 (Output)
*   각 항목별 획득 점수
*   **총 가점 (Total Score)**
